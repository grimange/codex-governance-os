# Codex Session Handoffs

This directory is the canonical root for governed Codex session handoff
packets.

Naming convention:

- `codex-session-handoff-{session-id}.md`

Canonical example:

- `codex-session-handoff-CS-20260314-001.md`

The governing contract for this artifact family is
[codex-session-handoff-packet-and-continuity-contract.md](../../../contracts/codex-session-handoff-packet-and-continuity-contract.md).

Packets in this directory are continuity records. They supplement but do not
replace:

- [codex-session-registry.md](../../../governance/codex-session-registry.md)
- [codex-session-ledger.md](../../../governance/codex-session-ledger.md)

No live handoff packets are required to exist at establishment time. Later
governed work may add append-only packet files here when a real handoff occurs.
