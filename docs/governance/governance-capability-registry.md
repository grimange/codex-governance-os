# Governance Capability Registry

This document is the canonical registry of governance capability state for
`codex-governance-os`.

It records each capability's current status, target state, implementing or
planned pipelines, and evidence basis. Capability state is evidence-backed and
must not be advanced without governed pipeline execution.
