## Introspection Boundaries

- This surface consolidates repository governance state. It does not compute new
  maturity values.
- Capability and layer states remain evidence-backed and subordinate to the
  canonical registry, progress, and maturity surfaces.
- Planned capability work must remain visibly unimplemented until later governed
  pipelines establish and verify it.
- This surface exposes documentation-governed state, not live runtime state.
