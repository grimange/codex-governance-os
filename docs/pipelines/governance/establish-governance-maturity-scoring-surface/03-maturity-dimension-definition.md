# Maturity Dimension Definition

## Governance Execution

Definition:

Measures how much recorded governance history resolves into positive execution
families recognized by the verdict canon.

Signals:

- `IMPLEMENTED`
- `ESTABLISHED`
- `VERIFIED`
- `NORMALIZED`

## Governance Safety

Definition:

Measures the share of recorded governance history that is not presently bounded
by limitation or gap families.

Signals:

- `WITH_LIMITATIONS`
- `WITH_GAPS`
- `RESTRICTED`

## Governance Coverage

Definition:

Measures how complete centralized governance memory is for historical pipeline
activity.

Signals:

- backfilled historical run count
- explicit unresolved historical gap count

## Governance Intelligence

Definition:

Measures whether the repository has the deterministic interpretation stack
needed to support repeatable maturity reporting.

Signals:

- `pipeline-run-ledger.md` exists
- `pipeline-run-analytics.md` exists
- `verdict-family-classification-canon.md` exists
