# Layer 0 Missing Capabilities

## Missing Or Partial Surfaces

### Central Layer 0 Evidence Canon

- current state: `PARTIAL`
- missing element:
  - one canonical doctrine that defines what counts as proof, what counts only as interpretation, and how evidence-backed verdicts differ from summaries

### Central Layer 0 Safety Invariant Canon

- current state: `PARTIAL`
- missing element:
  - one repository-wide invariant surface for fail-closed behavior, no silent semantic mutation, no hidden unsupported boundary, and no self-authorizing governance mutation

### Central Normalization Boundary Canon

- current state: `IMPLICIT_ONLY`
- missing element:
  - a doctrine that distinguishes safe automatic normalization from blocking conditions and authority-sensitive mutation

### Canonical Claim Classification Taxonomy

- current state: `PARTIAL`
- missing element:
  - one canonical taxonomy for `established`, `verified`, `partial`, `implicit-only`, `restricted`, `unsupported`, `blocked`, and `advisory`

## Risk Summary

- Layer 1 and Layer 2 work can continue, but Layer 3 expansion would rely too heavily on distributed conventions rather than a fully centralized Layer 0 canon.
