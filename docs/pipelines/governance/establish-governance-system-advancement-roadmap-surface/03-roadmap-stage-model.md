# Roadmap Stage Model

Pipeline 170 organizes advancement into deterministic stages derived from
remediation strategy ordering.

Current stage mapping:

- `state_normalization`
  - remediation entries with strategy `state_normalization`
- `capability_expansion`
  - remediation entries with strategy `capability_completion` or
    `verification_missing`
- `evidence_gap_resolution`
  - remediation entries with strategy `evidence_gap`

Current generated stages:

1. `state_normalization`
