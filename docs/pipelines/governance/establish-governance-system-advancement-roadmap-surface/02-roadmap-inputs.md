# Roadmap Inputs

Pipeline 170 derives the roadmap from these canonical surfaces:

- `docs/governance/governance-system-maturity.json`
- `docs/governance/governance-system-gaps.json`
- `docs/governance/governance-system-gap-remediation-plan.json`

These inputs provide:

- current domain maturity classifications
- current blockers to full maturity
- ordered remediation entries and dependency order
