# Evidence Insufficiency And Conflict Boundary

The canon now defines explicit handling for the three failure modes that were
previously only implied:

- insufficient evidence
  - the claim must be narrowed and the missing proof recorded explicitly
- conflicting evidence
  - authority order resolves the conflict first; unresolved conflicts remain
    unresolved until reconciled in version control
- missing evidence
  - silence is not proof and success must not be inferred from surrounding
    narrative

These rules prevent final verdicts or summaries from silently outrunning the
repository's actual proof.
