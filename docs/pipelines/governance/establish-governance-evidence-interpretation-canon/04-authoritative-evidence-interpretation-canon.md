# Authoritative Evidence Interpretation Canon

The canonical Layer 0 surface established by this lane is:

- [governance-evidence-interpretation-canon.md](/home/ramjf/python-projects/codex-governance-os/docs/governance/governance-evidence-interpretation-canon.md)

The doctrine defines the repository-wide interpretation rules that future
governance lanes must reuse:

- evidence must be scoped to the claim it actually supports
- evidence must respect authority order
- final verdicts must be evidence-backed and may not stand alone
- narrative must not override verification, registry truth, matrix truth, or
  explicit restrictions
- bounded evidence must preserve bounded claims explicitly
- tool output is authoritative only when the tool is canonical for the domain

This canon is Layer 0 because it governs how later layers interpret proof, not
just how one subsystem behaves.
