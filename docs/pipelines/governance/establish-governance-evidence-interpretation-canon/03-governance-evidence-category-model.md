# Governance Evidence Category Model

The new canon distinguishes four repository-wide categories:

- `authoritative evidence`
  - may directly support a governance claim when valid and in scope
- `supporting evidence`
  - may corroborate a claim but must not stand alone where stronger proof is
    required
- `narrative or descriptive material`
  - explains repository state but does not prove it by itself
- `non-evidence`
  - must not be treated as governance proof

This category model closes the Layer 0 ambiguity identified by pipeline
`L0-00`: evidence is no longer treated as a single undifferentiated class.
