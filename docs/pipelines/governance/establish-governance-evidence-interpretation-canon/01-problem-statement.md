# Problem Statement

Pipeline `L0-00` established that the repository already had a strong evidence
culture but not a centralized evidence interpretation canon. Evidence rules were
distributed across doctrine, artifact conventions, runtime tooling, tests, and
planning ledgers.

That left the repository behaving as if it had an evidence model without one
single Layer 0 doctrine stating what counts as proof, what only corroborates a
claim, and what must never stand alone.
