# Role Handoff Protocol

The established handoff protocol is deterministic:

1. Governance Planner selects the next valid governed path.
2. Surface Integrator aligns the required canonical entry surfaces.
3. Governance Verifier inspects structural sufficiency and evidence presence.
4. Evidence Reporter records the bounded verdict and execution evidence.

Continuous oversight during the workflow:

- Restriction Keeper preserves fail-closed boundaries throughout all steps.
- Architecture Auditor protects doctrine integrity when the work affects
  architecture-sensitive surfaces.

The protocol is collaborative but not orchestration-claiming. It defines how
roles conceptually interact during governed work without claiming runtime
sub-agent coordination.
