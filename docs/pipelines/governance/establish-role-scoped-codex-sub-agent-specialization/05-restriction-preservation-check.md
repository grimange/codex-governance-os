# Restriction Preservation Check

This lane preserves the inherited restrictions from Layers 0 through 3:

- no authority broadening beyond lower layers
- no universal or autonomous multi-agent claim
- no runtime orchestration enforcement claim
- no replacement of reusable skills or current single-agent execution behavior

The role model also adds its own explicit restriction set:

- role specialization is doctrinal only in this lane
- unsupported conditions still fail closed
- safe escalation requires reporting the boundary and recommending the next safe
  governed path
