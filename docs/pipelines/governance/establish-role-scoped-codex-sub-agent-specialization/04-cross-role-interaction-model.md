# Cross-Role Interaction Model

The canonical conceptual interaction order is:

- planner classifies and routes
- integrator maintains discoverability surfaces
- verifier inspects evidence and structure
- restriction keeper preserves fail-closed boundaries
- architecture auditor guards architecture alignment when relevant
- evidence reporter records bounded outcomes

This interaction model is governance doctrine only. It does not claim runtime
multi-agent execution. It defines how roles relate conceptually so later
specialization lanes can harden or verify behavior safely.
