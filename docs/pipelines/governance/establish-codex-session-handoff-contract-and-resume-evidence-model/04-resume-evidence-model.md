# Resume Evidence Model

Canonical resume admissibility rules now require:

- explicit predecessor handoff packet linkage
- bounded subject matching or narrowing predecessor scope
- preserved restrictions carried forward without silent weakening
- unsupported boundaries kept explicit
- evidence references sufficient to reconstruct current truth
- an admissible next action under governing doctrine
- no upgrade of pending or unverified claims into fact

Fail-closed interpretation:

- if those conditions are incomplete, the repository must prefer fresh
  initiation, interrupted execution, incomplete handoff, or resume inadmissible
  pending evidence rather than silently claiming continuity
