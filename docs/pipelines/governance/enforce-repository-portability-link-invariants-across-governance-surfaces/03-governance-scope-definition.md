# Governance Scope Definition

This invariant applies to active governed repository surfaces, including:

- canonical governance doctrine under `docs/governance/`
- canonical contracts under `docs/contracts/`
- repository entry surfaces such as `README.md`
- Codex discovery and operating surfaces such as `.codex/AGENTS.md`
- active pipeline definitions under `docs/pipelines/governance/*--*.md`
- active registry truth under `docs/pipelines/registry/pipeline-registry.md`

This lane preserves the historical evidence rule from pipeline `109`:

- historical artifact bundles may still contain earlier machine-local links as
  recorded defect evidence
- those historical mentions do not weaken the invariant for active canonical
  surfaces
- repository-wide cleanup of all older artifact bundles remains future work,
  not a prerequisite for establishing the invariant
