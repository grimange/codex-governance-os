# Capability Registry Design

Pipeline 160 establishes the canonical registry at:

`docs/governance/governance-capability-registry.md`

Registry model:

- `Capability`
- `Status`
- `Target State`
- `Implementing Or Planned Pipelines`
- `Evidence`

Status classes used in this lane:

- `complete`
- `in progress`
- `planned`

The registry acts as the single canonical surface for governance capability
state.
