# Expanded Supported Matrix

The supported matrix verified by pipeline `037` is:

- base-only
- `django + monorepo`
- `service + monorepo`
- `node-typescript-service + monorepo`
- `node-typescript-service + cli-worker`
- `cli-worker + monorepo`
- `cli-worker + python-package`
- `cli-worker + php-package`

Each of these compositions was re-verified across contract, doctor, scaffold, and inventory surfaces.
