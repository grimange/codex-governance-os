# Problem Statement

The repository now has a verified centralized pipeline ledger, but no dedicated analytics surface exists to answer governance behavior questions quickly.

Pipeline 143 establishes a canonical derived report that answers:

- how many runs are recorded and what the latest run is
- what kinds of pipelines are being executed over time
- verdict patterns and limitation/ gap signals
- recent governance activity context
- historical backfill coverage status

This surface is non-authoritative and must remain a pure derivation from `docs/governance/pipeline-run-ledger.md`.
