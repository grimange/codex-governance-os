# 03 — Verdict Family Taxonomy

The canonical families are:

- `VERIFIED`
- `ESTABLISHED`
- `NORMALIZED`
- `IMPLEMENTED`
- `WITH_LIMITATIONS`
- `WITH_GAPS`
- `RESTRICTED`
- `OTHER`

`OTHER` is a bounded fallback for legacy verdicts that do not match a canonical family token.

`WITH_LIMITATIONS` indicates a bounded limitation while preserving successful execution.
