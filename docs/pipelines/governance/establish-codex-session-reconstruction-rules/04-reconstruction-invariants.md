# Reconstruction Invariants

The reconstruction rules preserve the following invariants:

- state-machine authority remains sole authority for lifecycle semantics
- ledger integrity remains intact and reconstruction must not invent events
- registry identity authority remains intact through canonical `session_id`
- lifecycle observation remains interpretive and subordinate
- runtime neutrality is preserved and runtime context remains supporting only

The doctrine also preserves fail-closed handling when evidence is incomplete or
conflicting.
