# Ambiguity And Stale-Evidence Boundaries

## Ambiguity Handling

Ambiguity is handled materially, but not through a dedicated subsection.

Current coverage:

- conflicting evidence must resolve through authority order first
- unresolved conflicts remain unresolved until reconciled in version control

Interpretation restriction:

- the canon does not yet define a dedicated ambiguity doctrine section by name

## Stale-Evidence Handling

Stale evidence is handled materially, but not through a dedicated subsection.

Current coverage:

- stale or superseded summaries are classified as non-evidence

Interpretation restriction:

- the canon does not yet define a dedicated stale-evidence handling section by
  name

## Incomplete Evidence

Incomplete evidence is explicitly handled:

- insufficient evidence requires narrowing the claim and recording the missing
  evidence
- missing evidence may not be interpreted as success
- `Silence is not proof.`

Result: `BOUNDARIES_PRESENT_WITH_RESTRICTIONS`
