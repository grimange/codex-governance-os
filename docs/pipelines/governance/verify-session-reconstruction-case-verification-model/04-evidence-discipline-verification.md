# Evidence Discipline Verification

The case-model canon requires explicit evidence discipline.

Verification findings:

- every case must declare all evidence used during reconstruction evaluation
- evidence examples are explicit and bounded:
  - conversation transcripts
  - pipeline artifacts
  - governance doctrine
  - verification logs
  - referenced governance documents
- the canon states that all evidence used by the case must be explicitly cited
- the canon states that uncited evidence must not influence the case outcome
- the canon requires assumptions, inference boundaries, and ambiguity markers
  to remain explicit whenever evidence gaps exist

This preserves citation fidelity and prevents implicit evidence from shaping
case outcomes.

## Classification

- explicit evidence declaration required: `VERIFIED`
- uncited evidence permitted: `NOT OBSERVED`
- assumption transparency preserved: `VERIFIED`
- implicit assumption embedding permitted: `NOT OBSERVED`
