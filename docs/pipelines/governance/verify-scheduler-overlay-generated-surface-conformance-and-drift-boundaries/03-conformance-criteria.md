# Conformance Criteria

Scheduler generated surfaces pass this lane only if all of the following hold:

- generated file content matches the canonical scaffold renderer exactly
- each generated file includes explicit markers:
  - `# GENERATED BY TEMPLATE: scheduler-overlay`
  - `# DO NOT MODIFY GENERATED REGION DIRECTLY`
  - `# BEGIN GENERATED REGION`
  - `# END GENERATED REGION`
- each generated file exposes a post-generated custom extension boundary
- mutations inside the protected region are distinguishable from canonical truth
- custom extensions appended outside the protected region preserve the protected region
- supported scheduler compositions remain admitted and unsupported controls remain fail-closed
