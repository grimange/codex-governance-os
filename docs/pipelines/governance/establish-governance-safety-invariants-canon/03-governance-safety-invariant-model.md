# Governance Safety Invariant Model

The new canon defines six top-level invariant families:

- governed execution invariants
- evidence-scoped claim invariants
- canonical surface protection invariants
- restriction preservation invariants
- authority precedence invariants
- semantic safety invariants

This model closes the Layer 0 gap identified in pipeline `L0-00`, where
fail-closed behavior existed but did not yet have one centralized doctrinal
surface.
