# Claim Scope And Restriction Preservation Canon

The canon now explicitly protects against the claim-safety failures named in the
pipeline definition:

- claims must not exceed evidence scope
- verdicts and summaries must not outrun evidence
- local proof does not imply global safety
- restrictions must remain explicit
- unsupported remains its own governance state
- fail-closed boundaries must stay visible

This extends the evidence interpretation canon by stating not only how evidence
is read, but what governance must never silently broaden or erase.
