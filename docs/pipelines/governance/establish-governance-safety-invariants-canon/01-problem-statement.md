# Problem Statement

The repository already had strong fail-closed behavior in practice, especially
across composition contracts, evidence interpretation, and governance mutation
discipline. But those safety rules were still distributed across subsystem
contracts, doctrine, tests, and local agent guidance.

Pipeline `L0-00` identified the missing central Layer 0 safety invariant canon
explicitly. This lane centralizes the non-negotiable rules that must remain
true regardless of convenience, speed, or narrative confidence.
