# Template Family Definition

The universal template system now defines these canonical families:

- pipeline
- verification
- rule
- skill
- evidence-pack
- decision
- remediation

Each family has:

- a canonical markdown template under `docs/governance/templates/`
- required sections and optional sections in the registry
- required frontmatter fields
- validator and scaffold mapping
- overlay compatibility definitions

These families satisfy the minimum set required by the governing pipeline.
