# Restrictions And Non-Claims

This first Layer 3 canon is established with explicit restrictions.

Restrictions:

- it does not claim runtime enforcement of every Layer 3 rule
- it does not establish universal multi-agent or collaboration governance
- it does not resolve the remaining Layer 0 and Layer 1 restrictions
- it does not claim full coverage for governance domains not yet implemented in
  this repository
- it remains bounded to Codex behavior inside `codex-governance-os`

Non-claims:

- this lane does not verify Layer 3
- this lane does not certify that Codex is automatically compliant merely
  because the canon now exists
- this lane does not replace later request-classification or safe-mutation
  specialization lanes
