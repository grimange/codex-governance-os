# Layer 3 Role Definition

Layer 3 is the Codex operating layer.

Its role is to translate the lower governance layers into explicit Codex
behavior for:

- request classification
- governed routing
- safe mutation
- evidence-bounded authorship
- verification posture
- remediation posture
- bounded reporting

Layer 3 is subordinate to:

- Layer 0 doctrine and safety canons
- Layer 1 interpretation canon
- Layer 2 execution and registry truth

It does not replace those layers. It operationalizes them for Codex-directed
work.
