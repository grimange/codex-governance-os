# Codex Rule Domain Inventory

The established Layer 3 canon covers the required rule domains:

- authority order and canonical sources
- request classification
- governed routing
- safe mutation and artifact editing
- evidence-bounded authorship
- verification posture
- remediation posture
- unsupported, ambiguous, and stale-boundary handling
- reporting and recommendation discipline
- discoverability and operator usability

These domains were derived from the pipeline definition and from the currently
distributed Codex-facing rule surfaces already present in the repository.
