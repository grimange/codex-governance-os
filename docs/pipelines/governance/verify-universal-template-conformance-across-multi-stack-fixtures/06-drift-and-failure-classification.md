# Drift And Failure Classification

## Supported Fixtures

No supported fixture exposed:

- universal template leakage
- governance metadata inconsistency
- overlay/base contradiction
- realization harness defects

## Unsupported Fixtures

The unsupported categories classify as:

- `missing overlay`: Node or TypeScript service fixture
- `missing overlay`: CLI or worker-oriented fixture

These are explicit current support-boundary gaps, not failures in the supported matrix.
