# Restrictions And Non-Claims

The following restrictions remain explicit and prevent a full-closure verdict:

- no centralized Layer 0 normalization-boundary canon yet exists
- no centralized governed claim-classification taxonomy yet exists
- the evidence canon was verified only with restrictions because ambiguity and
  stale-evidence handling remain implicit rather than having dedicated
  subsections

Non-claims preserved by this lane:

- this lane does not claim runtime enforcement beyond what other lanes already
  established
- this lane does not claim Layer 1, Layer 2, or Layer 3 closure
- this lane does not claim that all doctrine gaps from `L0-00` are now closed
- this lane does not treat documentation presence alone as proof of enforcement

Because these restrictions are real, Layer 0 must not be promoted as fully
complete.
