# Enforcement Scope

Documented enforcement now applies to:

- governed Codex sessions
- sessions executing governance pipelines
- sessions mutating governance artifacts
- sessions handing work to another session
- sessions closing after in-scope governed execution that requires continuity
  evidence

Documented enforcement does not apply to:

- ad hoc exploratory work outside governed execution
- external tools outside repository-governed session operation
- runtime automation beyond what the repository explicitly implements later
