# Execution Ledger Model

The canonical execution ledger is established at
[codex-session-ledger.md](/home/ramjf/python-projects/codex-governance-os/docs/governance/codex-session-ledger.md).

Established ledger elements:

- lifecycle transition event vocabulary
- orchestrator-session and agent-role attribution
- task-scope and mutation-scope recording
- explicit handoff event fields
- final verdict recording for closure-oriented events

The ledger begins as an empty canonical table until later governed work records
specific session events.
